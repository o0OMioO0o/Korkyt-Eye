## Korkyt Eye: Система мониторинга поведения студентов

Проект по автоматизации контроля дисциплины в учебных аудиториях с использованием компьютерного зрения (Computer Vision).

## О проекте
Система разработана для автоматического определения действий студентов в аудитории в режиме реального времени. Проект помогает оперировать объективными данными о вовлеченности студентов в учебный процесс.

## 🛠 Технологии
* **Python**
* **YOLOv8** (архитектура для детекции)
* **OpenCV** (обработка видеопотока)
* **Roboflow** (работа с датасетом)

## Функционал:
Система распознает следующие классы действий:
- **Sleeping** (Сон)
- **Usingmobile** (Использование телефона)
- **Eating** (Прием пищи)

При фиксации нарушения система автоматически сохраняет кадр с доказательством (`PROOF_...jpg`) и выводит уведомление в консоль.
